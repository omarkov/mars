\chapter{Einleitung}
\section{Das Projekt Mar-S}
Das Projekt Mar-S wird durchgeführt, um eine Steuerung für den Sinnesraum am OIC zu entwickeln. 
Ziel der Software ist es in Umgebungen gleicher Ausstattung wieder verwendet zu werden.
Da immer wieder neue Komponenten hinzukommen können, muß gewährleistet sein, daß diese mit minimalem Aufwand hinzugefügt werden können. Also ohne direkt den Code zu ändern.
In der ersten Version besteht der Sinnesraum und dessen zu entwickelnde Software aus folgenden Komponenten:
\begin{itemize}
\item PDA-Steuerung
\item Weboberfläche für Einstellungen (Admin + User)
\item Konzept und Umsetzung der Datenhaltung für Mediadaten
\item Office Positioning System 
\item Datenbank 
\item Webserver 
\item Lichtsteuerung 
\item Windows XP MediaCenter Edition
\end{itemize}
