\chapter{Mediacenter}

\section{Übersicht}
Die Mediacenterkomponente MCConnect ist zu unterteilen in einen sichtbaren Teil,
das Interface, das direkt zum Kontrollieren der externen Module vom Mediacenter
aus angesteuert werden kann, und einen  Teil, welcher im Hintergrund
um Nachrichten vom Controller zu empfangen oder an ihn zu senden.

\section{Background MCConnect}
Wird durch ein Backgroundplugin für den Mediacenter realisiert, das beim Start der 
Nutzerumgebung geladen wird, die externen Module im MCConnect registriert und 
im Hintergrund weiterläuft um die folgenden Funktionen ermöglichen zu können.

\subsubsection{Funktionen}
\begin{itemize}
\item Empfangen von Nachrichten bezüglich Änderungen der Parameter externer Module.
\item Registrieren der sich anmeldenden externen Module im MCConnector zur 
      Darstellung im Interface.
\item Empfangen von Nachrichten zum Abspielen von Medien (zB.: Playlist).
\item Senden von Nachrichten an den Controller oder andere externe Module 
      bezüglich Parameteränderungen oder ähnlichen Modifikationen an der 
      Mar-s Umgebung.
\end{itemize}

\section{Interface}
Wird mit dem laden eines Profils abhängig von den im MCConnect registrierten 
externen Modulen generiert. Die Mar-s Systemeinstellungen, realisiert durch den 
MCConnect sind über einen Menüpunkt im Hauptmenu des Mediacenters erreichbar.

\subsection{Interface Aufbau}
Das Interface besteht aus einer Combobox, welche der Auswahl des einzustellenden 
Moduls dient. Nach der Auswahl des Moduls werden dessen registrierte Parameter zur
Modifikation angezeigt, welche grundsätzlich Booleans, Zahlenwerten (numerische 
Parameter), Zeichenketten oder einer Liste aus Zeichenketten sein können.

\subsection{Visualisierung der Parameter}

\begin{description}
\item [Boolscher Parameter:]Checkbox mit Namen des Parameters
\item [Numerischer Parameter:]Slider mit Name des Paramters
\item [Zeichenketten:]Textbox
\item [Liste aus Zeichenketten:]Listbox
\end{description}
