\section{Web-Oberfläche}

\subsection{Die Administratoransicht}

Ist man im System eingeloggt und besitzt Administratorrechte hat man die Möglichkeit Benutzer, Abteilungen und Gruppen anzulegen, zu verändern und zu löschen (siehe Abb~\ref{hauptAdmin}). Im Folgenden werden diese einzelnen Punkte genauer beschrieben.

\begin{figure}[h!]
\includegraphics[width = 450pt]{pix/hauptAdmin.jpg}
\caption{Administratoransicht}
\label{hauptAdmin}
\end{figure}

\newpage

\subsection{Verwaltung der Systembenutzer}

In diesem Abschnitt wird beschrieben, wie man die Benutzer des Systems verwaltet, d.h. neue Benutzer anlegen, Benutzer löschen oder Benutzer editieren kann. Hierzu klickt man in der Administratoransicht (siehe Abb.~\ref{hauptAdmin}) im Menü auf ''Benutzer''. Es erscheint die Seite (siehe Abb.~\ref{user}) um Benutzer zu verwalten.

\begin{figure}[h!]
\includegraphics[width = 450pt]{pix/user.jpg}
\caption{Benutzerfenster}
\label{user}
\end{figure}


\newpage

\subsubsection{Erstellen eines neuen Benutzers}

\begin{enumerate}

\item In der Administratoransicht auf ''Benutzer'' klicken
\item Auf ''Neuer Benuzer'' klicken. Es erscheint eine neue Seite (siehe Abb.~\ref{userNew})
\item Daten für den Benutzer eingeben. Benutzername ist der Loginname im System. Die Quota legt fest wieviel Speicherplatz dem Benutzer für seine Daten zu Verfügung stehen. Setzt man die Quota auf 0, so ist diese unbegrenzt.
\item Sind alle Eingaben eingetragen, so wird der Benutzer mit ''Speichern'' in die Datenbank übernommen. Mit ''Reset'' werden die Eingaben rückgängig gemacht.

\end{enumerate}

\begin{figure}[h!]
\includegraphics[width = 450pt]{pix/userNew.jpg}
\caption{Neuer Benutzer}
\label{userNew}
\end{figure}

\newpage

\subsubsection{Editieren eines vorhandenen Benutzers}

\begin{enumerate}

\item In der Administratoransicht auf ''Benutzer'' klicken
\item Auf ''Edit'' klicken. Es erscheint eine neue Seite (siehe Abb.~\ref{userEdit}).
\item Änderungen des Benutzers vornehmen
\item Sind alle Eingaben eingetragen, so wird der Benutzer mit ''Speichern'' in die Datenbank übernommen. Mit ''Reset'' werden die Eingaben rückgängig gemacht.
\end{enumerate}

\begin{figure}[h!]
\includegraphics[width = 450pt]{pix/userEdit.jpg}
\caption{Benutzer editieren}
\label{userEdit}
\end{figure}

\newpage

\subsubsection{Löschen eines Benutzers}

\begin{enumerate}

\item In der Administratoransicht auf ''Benutzer'' klicken
\item Auf ''Delete'' klicken.
\item Es folgt noch eine Sicherheitsabfrage (siehe Abb.~\ref{userDelete}), dann wird der Benutzer gelöscht.

\end{enumerate}

\begin{figure}[h!]
\includegraphics[width = 450pt]{pix/userDelete.jpg}
\caption{Benutzer löschen}
\label{userDelete}
\end{figure}

\subsubsection{Editieren einer vorhandenen Gruppe}

\begin{enumerate}
\item In der Administratoransicht auf ''Gruppen' klicken
\item Auf ''Edit'' für die zu editierende Gruppe klicken. Es erscheint eine neue Seite (siehe Abb.~\ref{gruppeEdit}).
\item Änderungen für die Gruppe eingeben. Links bei den aufgelisteten Benutzern werden die Markierungen gesetzt. Um die Benutzer zu entfernen oder hinzuzufügen sollten die Buttons ''Benutzer hinzufügen'' und ''Benutzer löschen'' noch vor dem ''Speichern''- Button geklickt werden und die entsprechenden Benutzer markiert werden.
\item Sind alle Eingaben eingetragen, so wird die Gruppe mit ''Speichern'' in die Datenbank übernommen. Mit ''Reset'' werden die Eingaben rückgängig gemacht.

\subsubsection{Email an alle Gruppenmitglieder senden}

\begin{enumerate}
\item In der Administratoransicht auf ''Gruppen' klicken
\item Auf ''E-Mail absenden'' klicken. Es erscheint eine neue Seite (siehe Abb.~\ref{gruppeMail}).
\item Betreff und Nachricht eingeben.
\item Alle Gruppen markieren, an die die E-Mail geschickt werden soll
\item  Mit ''E-Mail absenden'' wird die E-Mail verschickt. Mit ''Reset'' werden die Eingaben rückgängig gemacht.
\end{enumerate}
