\section{Einleitung}

Dies ist das Handbuch für das System ''Mar-S'', das im Rahmen eines Studienprojektes des Studiengangs Softwaretechnik der Universität Stuttgart entwickelt wurde. Dabei handelt es sich um ein System mit dem der adaptive Raum des Office Innovation Centers gesteuert und für verschiedene Benutzer eingestellt werden kann.\\

Das System gliedert sich in mehrere Module:

\begin{itemize}

\item Die Web-Oberfläche. Sie wird benutzt, um die Einstellungen im Sinnesraum vorzunehmen, d.h. Benutzer und deren Profile anlegen oder editieren.

\item Das MediaCenter (Freevo). Es läuft im Sinnesraum und wird zur direkten Steuerung der Einstellungen im Sinnesraum genutzt,  d.h. abspielen von Playlisten (Video, Musik, Bilder), Lichteinstellungen (Farbe, Helligkeit). Desweiteren können die wichtigsten Komponenten auch mit einem PDA oder einer Fernbedienung im Sinnesraum verändert werden.

\end{itemize}

\subsection{Verwaltung von Profilen}

In diesem Abschnitt wird beschrieben, wie man Profile eines Benutzers im System verwaltet; das heißt neue Profile anlegen, Profile löschen oder Profile editieren. Hierzu klickt man in der Benutzeransicht (siehe Abb.~\ref{hauptNoAdmin}) im Menü auf ''Meine Profile''. Es erscheint die Seite (siehe Abb.~\ref{profile}) um Profile zu verwalten.\\

Hier werden alle vorhandenen Profile für den Benutzer angezeigt. Jedes Profil kann als Default-Profil gesetzt werden. Es kann nur ein Default-Profil geben. Das Default-Profil wird beim Betreten des Sinnnesraumes automatisch gestartet. Profile enthalten die Einstellungen aller ansprechbarer und registrierten Komponenten im Sinnesraum (z.B. Lichteinstellungen, Musikplaylisten, usw.).\newline

\subsubsection{Erstellen eines neuen Profilen für einen Benutzer}

\begin{enumerate}
\item In der Benutzeransicht auf ''Meine Profile' klicken.
\item Auf ''Neues Profil'' klicken. Es erscheint eine neue Seite (siehe Abb.~\ref{profileNew}).

\item Sind alle Angaben eingetragen, so wird das Profil mit ''Speichern'' in die Datenbank übernommen. Mit ''Reset'' werden die Eingaben rückgängig gemacht.
\end{enumerate}

\section{Bedienung im Sinnesraum}

In diesem Abschnitt wird beschreiben, wie man im Sinnesraum als Benutzer seine Profile abruft und die aktuellen Komponenteneinstellungen verändert, dabei ist zu beachten, dass diese Änderungen nicht gespeichert werden. D.h. falls z.B. die Lichtfarbe auf ''Rot'' im aktuellen Profil gesetzt ist und sie wird im Sinnesraum auf ''Grün'' gestellt, wird dies in Echtzeit übernommen, jedoch nicht in der Datenbank gespeichert.\\

Die Steuerung im Sinnesraum geschieht im Wesentlichen über drei Module:

\begin{itemize}

\item Das Mediacenter Freevo
\item Ein PDA
\item Eine Fernbedienung
\end{itemize}

\subsection{Freevo}

Unter ''Freevo'' hat man die Möglichkeit seine eigenen Profile auszuwählen, den eingeloggten Benutzer zu wechseln oder einfach nur seine eigenen Dateien anzuschauen (Videos und Bilder) oder anzuhören (Musik). Dies geschieht über das Freevo--Hauptfenster (siehe Abb.~\ref{FreevoMain}).

\begin{enumerate}
\item \emph{''Profiles''} Man wählt das zu spielende Profil aus. Ein Beispiel siehe Abb.~\ref{FreevoProfile}.
\item \emph{''Change User''} Es werden alle sich im Sinnesraum befindenden Benutzer angezeigt. Jeder dieser Benutzer lässt sich  einloggen. Es lässt sich nur ein Benutzer gleichzeitig einloggen. Die Einstellungen des eingeloggten Benutzers(Lich, Musik usw.) werden geladen und abgespielt.
\item \emph{''Watch Movie''} Man kann einen speziellen Film anschauen, die nicht unbedingt in einem Profil abgelegt sein müssen.
\item \emph{''Listen to Music''} Man kann spezielle Musiktitel abspielen, die nicht unbedingt in einem Profil abgelegt sein müssen.
\item \emph{''Look at Pictures''} Man kann sich spezielle Bilder anschauen, die nicht unbedingt in einem Profil abgelegt sein müssen.
\end{enumerate}

\subsection{Fernbedienung im Sinnesraum}

\begin{figure}
\includegraphics[width = 400pt]{pix/Fernbedienung.jpg}
\caption{Microsoft Media Center Fernbedienung}
\label{Fernbedienung}
\end{figure}

Mit der Fernbedienung (siehe Abb.~\ref{Fernbedienung}) kann durch das gesamte Freevo-Menu navigiert werden. Ebenso ist es möglich direkt Lautstärke, Lichteinstellungen und Stimmungen (das sind spezielle, voreingestellte Lichteinstellungen) zu steuern.\\[1ex]

\textbf{Lautstärke steuern}

\begin{enumerate}

\item Erhöhung der Lautstärke \emph{Volume} ''+'' drücken.
\item Veringerung der Lautstärke \emph{Volume} ''-'' drücken.

\end{enumerate}

\textbf{Stimmungswechsel}

\begin{enumerate}

\item Es existieren 5 Stimmungen, die man dirket mit der entsprechenden Ziffer aufrufen kann.

\end{enumerate}
\newpage

\textbf{Lichtsteuerung}

\begin{enumerate}

\item Auf den farbigen Tasten (''Rot'', ''Blau'', ''Grün'', ''Gelb'') stellt man die jeweilige Farbenintensität ein.
\item Taste ''Rot'' drücken.
\item Durch ''+'' und ''-'' wird die Lichtintensität für ''Rot'' geändert. Es ist auch möglich die Intensität durch dirkete Angabe einer Zahl zu verändern.
\item Mit der  Taste ''Eingabe'' kommt man wieder in der Ausgangsposition. Es kann jetzt eine andere Farbe, z.B. ''Grün'' gewählt werden und wie oben beschrieben eingestellt werden.
\item Mit der Taste ''Löschen'' wird das Licht abgschaltet.

\end{enumerate}

\textbf{Freevo}

Freevo wird ganz intuitiv mit der Fernbedienung gesteuert.

\begin{enumerate}

\item Mit den Navigationstasten (''hoch'',''runter'', ''links'', ''rechts'' ) navigiert man durch die Menus.
\item Mit der ''OK'' --Taste bestätigt man seine Auswahl.
\item Mit der ''Zurück'' --Taste gelangt man einen Schritt zurück.

\end{enumerate}
